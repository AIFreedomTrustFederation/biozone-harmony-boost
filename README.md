## Purpose

`biozone-harmony-boost` is a concept-stage interface for exploring Circleunchain as the system for Biozoe Coin, AetherCoin, future biozoe coins, ecological stewardship dashboards, participation UI flows, and API shapes connected to the larger AI Freedom Trust Federation ecosystem.

This repository is a live public web service for the concept interface. Biozoe Coin, AetherCoin, and other future biozoe coins will be built on the Circleunchain system. Biozoe is the opposite of crypto; Circleunchain is the opposite of blockchain. The system must be built first. This is not a live token sale, wallet custody service, investment product, audited blockchain system, or production API. It is a research prototype that can inform future work in `Aether_Coin_biozonecurrency`.

## Setup

```bash

npm run dev
```

## Verification

```bash
npm run build
npm run lint
npm audit --audit-level=moderate
```
